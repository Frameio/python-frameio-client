frameioclient¶

Installation¶

Stable releases of frameioclient can be installed with

pip <http://pip.openplans.org> <- or you may download a `.tgz` source

archive from pypi. See the frameioclient page for more detailed instructions.

If you want to use the latest code, you can grab it from our Git repository, or fork it.
